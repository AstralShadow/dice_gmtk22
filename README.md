## Building
You basicly want to do:

    git clone https://github.com/AstralShadow/dice_gmtk22
    cd dice_gmtk22
    mkdir build
    cd build
    cmake ..
    make -j$(nproc)
    cp -r ../assets .
    ./dice

Current version is an alpha version.
